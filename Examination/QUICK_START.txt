================================================================================
QUICK START GUIDE - Dream Villa VR Experience
================================================================================

IMMEDIATE STEPS TO GET STARTED:

1. OPEN THE APPLICATION:
   - Open "index.html" in a modern web browser (Chrome/Edge recommended)
   - The scene will load automatically
   - Wait for the villa model to load (may take a few seconds)

2. NAVIGATION:
   - Use WASD keys or Arrow keys to move
   - Click and drag mouse to look around
   - Click the VR button (top right) to enter VR mode if you have a headset

3. EXPLORE:
   - Walk around the villa
   - Visit the three rooms: Living Room, Bedroom, Kitchen
   - Check out the animated swimming pool
   - See the trees, car, and pet dog

4. FOR DEPLOYMENT:
   - Upload entire "Examination" folder CONTENTS to your web server
   - Ensure all .glb files are in the same folder as index.html
   - Access via: https://YourIndexNumber.ceiscy.com/Examination/

================================================================================
TROUBLESHOOTING:
================================================================================

Problem: Villa model not loading
Solution: Ensure modern_villa_for_sale.glb is in the same folder as index.html

Problem: Low frame rate
Solution: Close other browser tabs, use Chrome/Edge, disable browser extensions

Problem: Controls not working
Solution: Click on the scene first to enable pointer lock, check browser console for errors

Problem: VR mode not available
Solution: Use Chrome/Edge, ensure WebXR is enabled in browser settings
